== Tic Tock Swipe the Timeclok

This is a simple timeclock application. It is currently being refactored to work with AJAX for faster performance and AJAX fun. Authentication is manually created. No devise this time!

Ruby 2.2.0p0 - Rails 4.2.0
Postgresql

Install:
Make sure you have the ruby and rails gems installed
Clone the repo 
$ bundle install
$ rake db:create
$ rake db:migrate
$ rails s
browser -> localhost:3000
